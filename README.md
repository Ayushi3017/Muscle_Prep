Welcome to the official frontend repository for **Muscle Prep**, a premium meal prep service offering protein-rich breakfast sample kits designed by nutritionists and athletes. 

This repository contains the codebase for our interactive landing page and ordering system.

## ✨ Key Features

* **Interactive Menu:** Clean, grid-based menu showcasing high-quality meal photography and price tags.
* **Dynamic Macro Tracking:** Clickable meal cards that reveal a hidden, animated `Chart.js` pie chart displaying the exact breakdown of Proteins, Carbs, and Fats.
* **Modern UI/UX:** Fully responsive design built with Tailwind CSS, ensuring a seamless experience on mobile, tablet, and desktop.
* **Order System:** User-friendly checkout forms with dynamic dropdowns for meal quantities and a UI toggle for UPI vs. Card payments.
* **Pre-Registration:** A waitlist signup form for future full-service meal plans.

## 🚀 Tech Stack

* **Structure:** HTML5
* **Styling:** Tailwind CSS (via CDN)
* **Data Visualization:** Chart.js
* **Icons:** FontAwesome 6.0
